## Infinite-Synthesis-x64
심심할때마다 밑바닥부터 만드는 운영체제 
* 개발 언어 : C, Assembly
* 개발 도구 : GCC Cross-Compiler, NASM, make, qemu 및 vmware

## OS Requirement
* 아키텍쳐 : x86_64
* 지원 메모리 : 64GB
* CPU : x86_64를 지원하는 모든 기종
* 메모리 : 64MB 이상 요구
* 하드디스크 : 파일 시스템 미지원

# Feature
* 기초적인 하드디스크 드라이버
* 파일 시스템
* 태스크 구현
* 라운드 로빈 스케줄러 구현
* 동적할당 구현
* 키보드 드라이버, 셸, 인터럽트 업그레이드
