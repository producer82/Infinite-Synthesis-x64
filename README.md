## Infinite-Synthesis-x64
Compile with gcc, nasm

## OS Requirement
* Architecture : x86_64
* Maximum support memory : 64GB
* Memory require : 64MB 이상 요구
* CPU : any processor that support x86_64

# Feature
* 기초적인 하드디스크 드라이버
* 파일 시스템
* 태스크 구현
* 라운드 로빈 스케줄러 구현
* 동적할당 구현
* 키보드 드라이버, 셸, 인터럽트 업그레이드
